# 코딩 전 사전 지식

### ATMEGA128의 Timer/Counter

#### Timer / Counter는 무엇인가?

타이머 카운터(Timer/Counter)는 마이크로컨트롤러에서 주어진 시간 동안의 카운트를 수행하는 하드웨어 모듈이다. 이 모듈은 내부적으로 클럭 신호를 카운트하고, 사용자가 설정한 카운터 값에 도달하면 인터럽트를 발생시키거나 외부 장치를 제어하는 등의 기능을 수행할 수 있다.

#### 왜 Timer / Counter를 사용하는가?

예를 들어 10ms마다 센서 값을 읽고 동시에 버튼 입력을 처리해야한다고 하자. \_delay()를 사용해서 구현을 하게된다면 "블로킹 지연"이 일어나기 떄문에 10ms동안 CPU가 멈추게 된다. 이는 버튼입력이 밀리거나 놓치게 되는 결과를 낳게 될 수 있다. 그러나 Timer/Counter 사용 시 하드웨어가 시간을 세고 인터럽트로 10ms마다 센서 읽기를 수행하므로, 메인 루프는 계속 버튼 처리가 가능하다.

#### 8Bit 타이머와 16Bit 타이머

**8Bit** 타이머는 0~255까지 카운트 하므로 빠른 주기, 짧은 시간 측정에 적합하다. 또한 설정이 간단하고 오버헤드가 적다
**16Bit** 타이머는 0~65535까지 카운트하므로 더 긴 시간 간격, 다양한 주기를 구현 가능하다.

하드웨어 자원(면적/전력/복잡도)을 줄이면서도 짧은 주기용 타이머와 긴 주기용 타이머를 각각 최적화하려는 설계 때문이다.
MCU에서는 필요한 타이밍 작업이 다양해서 짧은 주기 인터럽트는 8Bit로 충분하고, PWM 해상도/긴 타이밍은 16Bit가 필요하다.

#### 분주란 무엇인가?

ATMEGA128은 16MHz 클럭을 사용하며, 이는 1초에 1600만 번의 클럭 틱이 발생함을 의미한다. 이 속도로 타이머를 그대로 사용하면 매우 짧은 주기만 만들 수 있고, 긴 주기를 만들려면 지나치게 많은 카운트를 세어야 한다. 이런 문제를 줄이기 위한 개념이 **분주(Prescaler)**이다.

이 클럭을 쪼개기 위해서는 **분주비**라는 개념이 필요하다.
분주비란 입력 클럭을 N으로 나누어 타이머가 더 느린 속도로 카운트하도록 만드는 값이다.

기본적으로 1초에 1600만 번을 세는 ATMEGA128을 분주 없이 사용한다고 하면 1초를 세기 위해서는 카운팅이 1600만 번 필요하고 주기는 1/16000000 = 0.0000000625초가 된다.
여기서 **분주비** 개념을 이용해보자.
64 분주를 적용한다고 하면
주기는 64/16000000 = 0.000004초가 되고,
1초를 세는 데 필요한 카운팅은 250000회로 줄어들어, 기존의 1600만 번에서 크게 감소한다.

8비트 타이머를 사용한다면 최대 256번까지 카운팅 가능하고 64분주의 주기는 4us이므로 8비트 타이머가 64분주 주기를 256번 세면 1.024ms가 걸리게 되는 것을 알 수 있다.

코딩을 통해 250번만 세도록 설정한다면 비교적 정확한 1ms를 만들 수 있음을 알 수 있다.

#### 10ms 타이머 예시

```
/* 다음은 10ms마다 FND값을 증가시키는 코드이다.
다만, 해당 실제로는 10ms는 너무 짧은 시간이므로
FND값이 증가하는 것을 제대로 관측하긴 힘들것이다.
 CS12와 CS10으로 분주값을 1024로 두고 OCR1A를
 15624 로 설정하면 1초마다 값이 변하는 것을
  확인 가능할 것이다.*/

#define F_CPU 16000000L

#include <avr/io.h>
#include <avr/interrupt.h>
#include <avr/iom128.h>

unsigned char fnd_degit[] = {0x3F, 0x06, 0x5B, 0x4F, 0x66, 0x6D, 0x7D, 0x27, 0x7F, 0x6F};
volatile unsigned char count = 0;

ISR(TIMER1_COMPA_vect){
    count++;
	if (count>9){
		count=0;
	}
	PORTC=fnd_degit[count];
}

void timer1_init(void){
	TCCR1B |= (1<<WGM12) | (1<<CS11) | (1<<CS10); //64분주
	OCR1A=2499; //OCR1A값 지정
	TIMSK |= (1 << OCIE1A); //2500되면 -> 10ms되면 인터럽트
}

int main(void){
    DDRC=0xFF;
	DDRG=0x0F;
	PORTC=fnd_degit[0];
	PORTG=0x01;

    timer1_init();
    sei();
    while(1){

    }
}

```
